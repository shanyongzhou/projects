# 微信小程序－学富网

### 说明：

实现学富网(xuef.com)客户端部分功能，包括首页，课程和我的三个页面，特色：
- 课程页可拖动内容切换tab

### 数据接口:

使用本地数据

### 目录结构：

- image — 存放项目图片文件
- pages — 存放项目页面渲染相关文件
- utils — 存放日期处理功能代码

### 开发环境：

微信web开发者工具 v0.11.112301
